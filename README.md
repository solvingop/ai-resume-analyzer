AI Resume Analyzer & Interview Copilot
An AI-driven platform designed to bridge the gap between a candidate's resume and specific job requirements through deep analysis and personalized interview preparation.

🚀 Features
Tailored Technical Questions: Generates scenario-based questions derived from your specific project experience, such as architecting data syncs or handling "late data" in streaming pipelines.

Behavioral Interview Prep: Role-specific soft-skill questions based on your professional history.

Match Scoring: A percentage-based indicator (e.g., 98%) showing how well your profile aligns with a target role.

Skill Gap Identification: Highlights specific missing technical competencies, such as Vector Databases (Pinecone, Milvus) or Advanced Real-time UI frameworks.

Career Road Map: Provides a structured path to bridge identified gaps and prepare for the role.

🛠️ Tech Stack
Frontend: React, Tailwind CSS (Modern Dashboard UI)

Backend: Node.js, Express

Database: MongoDB (via Mongoose)

Authentication: JWT (JSON Web Tokens) with Secure HTTP-Only Cookies

AI Engine: LLM integration for resume parsing and question generation
